### Задание 3: Одинокая дорога
**Ответ:** 63,2; 93,3

**Кратко:**
Предположим пуассоновский поток: P(no car за 30 мин) = 0,05 = e^{-λ*30}.

P(10 мин) = 1 - e^{-λ*10} = 0,6316 → 63,2%.
P(27 мин) = 1 - e^{-λ*27} = 0,9325 → 93,3%.

## Блок 2: Python

### Задание 1: Изоморфизмы
```python
def is_isomorphic(s: str, t: str) -> bool:
    if len(s) != len(t):
        return False

    s_to_t = {}
    t_to_s = {}

    for cs, ct in zip(s, t):
        if cs in s_to_t and s_to_t[cs] != ct:
            return False
        if ct in t_to_s and t_to_s[ct] != cs:
            return False
        s_to_t[cs] = ct
        t_to_s[ct] = cs

    return True
```
**Сложность:**
Время O(n), память O(1) по алфавиту (в худшем O(n)).

### Задание 2: Натуральная последовательность
```python
def missing_number(nums: list[int]) -> int:
    n = len(nums) + 1
    expected = n * (n + 1) // 2
    actual = sum(nums)
    return expected - actual
```
**Сложность:**
Время O(n), память O(1).

### Задание 3: Факторизация
```python
def prime_factors(n: int) -> list[int]:
    factors = []
    d = 2
    while d * d <= n:
        while n % d == 0:
            factors.append(d)
            n //= d
        d += 1 if d == 2 else 2  # после 2 проверяем только нечётные
    if n > 1:
        factors.append(n)
    return factors
```
**Сложность:**
Время O(√n) в худшем случае, память O(1) кроме списка факторов.
